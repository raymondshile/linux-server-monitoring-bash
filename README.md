# Linux Server Monitoring (Bash)

A lightweight Bash-based monitoring tool designed to track system health, resource usage, and uptime on Linux servers.

---

## Overview

This project simulates real-world system administration tasks by monitoring CPU usage, memory consumption, disk space, and system uptime.

---

## Features

- CPU usage monitoring  
- Memory usage tracking  
- Disk space checks  
- System uptime reporting  

---

## Technologies Used

- Linux (RHEL / Ubuntu)  
- Bash scripting  
- Command-line tools (top, free, df, uptime)

---

## Use Case

This reflects real-world responsibilities of a system administrator or technical support engineer.

## How to Run

```bash
# Clone the repository
git clone https://github.com/raymondshile/linux-server-monitoring-bash.git

# Navigate into the project folder
cd linux-server-monitoring-bash

# Make the script executable
chmod +x monitor.sh

# Run the script
./monitor.sh
```
