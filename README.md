# Linux Server Monitoring (Bash)
## 🚀 Live Demo / Overview

This project simulates real-world Linux system monitoring by tracking CPU, memory, and disk usage, logging system performance over time, and triggering alerts when thresholds are exceeded.

It is automated using cron to mimic continuous monitoring in a production environment.

A lightweight Bash-based monitoring tool for tracking system health and performance.

## Features
- Real-time CPU, memory, and disk usage monitoring  
- Automated logging of system performance metrics  
- Threshold-based alerting system  
- Configurable monitoring via external config file  
- Automated execution using cron
  


  ## Real-World Application

This project simulates tasks performed by system administrators and technical support engineers in real production environments. 

It focuses on monitoring system health, identifying resource bottlenecks, and improving troubleshooting efficiency using native Linux tools.

  ## How to Run

```bash
# Clone the repository
git clone https://github.com/raymondshile/linux-server-monitoring-bash.git

# Navigate into the project folder
cd linux-server-monitoring-bash

# Make the script executable
chmod +x monitor.sh

# Run the script
./monitor.sh
```
